# projeto-calculadora
Este projeto simples, implementa uma calculadora com funções de soma, multiplicação, subtração,
divisão e exponenciação. Com a opção de deletar, calcular, deletar tudo e mais.

## Tecnologias ultilizadas.

- HTML
- CSS
- JavaScript

## O que aprendi durente o desenvolvimento?

- Estruturação de interface com HTML semântico.
- Estilização e organização visual utilizando CSS (layouts, botões e responsividade).
- Manipulação de eventos com JavaScript (click e teclado).
- Implementação da lógica de operações matemáticas (adição, subtração, multiplicação, divisão).
- Uso de DOM Manipulation para atualizar dinamicamente o visor da calculadora.
